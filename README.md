# Chrome 自动跳转插件

### 使用方法

- 克隆`代码仓库`到本地
<pre>git clone https://github.com/sqlnice/auto-goto-url.git</pre>

- 打开 `Chrome 扩展程序页面`
<pre>chrome://extensions/</pre>

- 打开右上角 `开发者模式`

- 点击 `加载已解压的扩展程序`, 选择文件夹即可
